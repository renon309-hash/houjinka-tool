# 法人化判定ツール 開発メモ

> 最終更新: 2026年2月28日

---

## 概要

個人事業主が法人化すべきタイミングを、税負担・手取りの比較で判定するシミュレーションツール。  
「所得700〜800万円で法人化」という曖昧な目安を、条件付きで数値化することが目的。

```

├── index.html          ← 本体（HTML + CSS + JS すべて含む単一ファイル）
├── 計算式一覧.md        ← 全計算式の数式ドキュメント

├── .gitignore
└── js/                  ← モジュール分離版（参考用・将来Node.js導入時に使用）
    ├── app.js
    ├── calculators/
    │   ├── personal.js       ← 個人事業の計算
    │   ├── corporate.js      ← 法人化ケースの総合計算
    │   ├── corporateTax.js   ← 法人税等（地方法人税含む）の計算
```

### 重要な注意

- **`index.html` が本番ファイル**。すべてのロジックがこの中に含まれている
- `js/` フォルダはモジュール分離版（ES Modules）で、参考・将来用。`.gitignore` で除外されておりGitHubにはプッシュされない
- 改修するときは **`index.html` 内の `<script>` タグ** を編集する

## 機能一覧（MVP）

### 入力項目

| 項目 | 種類 | 備考 |
|---|---|---|
| 年間売上 | 金額（カンマ区切り対応） | |
| 必要経費（年額） | 金額 | |
| 青色申告特別控除 | プルダウン | 65万/55万/10万/なし |
| 所得控除額（概算） | 金額 | 基礎控除・配偶者控除・社保控除等の合計 |
| 年齢 | 数値 | 40〜64歳で介護保険料加算 |
| 個人事業税の対象業種 | チェックボックス | |
| 役員報酬（年額） | 金額 | |
| 法人化による追加コスト（年額） | 金額 | 個人事業にはなく法人化で新たに発生する費用 |
| 代表が賃貸住宅に住んでいる | チェックボックス | 役員社宅制度の適用判定 |
| 月額家賃 | 金額 | 賃貸チェック時のみ表示。50%を法人経費に |
| 国保+国民年金（任意） | 金額 | 空欄なら自動概算 |
| 社保 本人負担（任意） | 金額 | 空欄なら自動概算 |
| 社保 会社負担（任意） | 金額 | 空欄なら自動概算 |

### 出力項目

**個人事業のまま:**
- 売上-経費、青色控除、事業所得、課税所得
- 所得税（復興特別所得税含む）、住民税、個人事業税、国保・国民年金
- 手取り

**法人化後（個人側）:**
- 役員報酬、給与所得控除、給与所得、課税所得
- 所得税、住民税、社会保険（本人負担）
- 役員社宅制度による実質手取り増（該当時）
- 個人の手取り

**法人化後（会社側）:**
- 売上-経費、役員報酬、追加コスト、社保（会社負担）、役員社宅（法人負担分・該当時）、税引前利益
- 法人税（国税）、地方法人税（国税）、法人住民税（法人税割・均等割）、法人事業税、特別法人事業税
- 法人税等合計、税引後利益（内部留保）
- 合算キャッシュ

**判定:**
- 個人手取りベースの差額
- 合算キャッシュベースの差額（内部留保含む）
- 3段階判定（有利/条件次第/まだ早い）
- 理由トップ3

## 計算ロジックの構造（index.html 内の関数）

```
getTaxTable(year)               ← 税率テーブル取得
calcIncomeTax(taxableIncome, t) ← 所得税の速算計算
calcPersonal(params)            ← 個人事業の全計算
calcSalaryDeduction(salary, t)  ← 給与所得控除
calcSocialInsurance(salary, age, year) ← 社保の概算計算
calcCorpTaxBase(income, table)  ← 法人税（本体）
calcCorpBusinessTax(income, brackets) ← 法人事業税（超過累進）
calcCorporateTaxes(profit, year) ← 法人税等の全内訳
calcCorporate(params)           ← 法人化ケースの総合計算
compare(personal, corporate)    ← 差額の算出
buildJudgement(personal, corporate, comp) ← 判定・理由生成
```
