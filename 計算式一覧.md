# 法人化判定ツール 計算式一覧

> 2026年度 MVP版の税率テーブルに基づく概算計算です。  
> 金額はすべて **円単位**、端数は **円未満切捨て（floor）** で統一しています。

---

## 1. 入力パラメータ

| 変数名 | 内容 |
|---|---|
| `sales` | 年間売上 |
| `expenses` | 必要経費（年額） |
| `blueReturnDeduction` | 青色申告特別控除（65万/55万/10万/0） |
| `deductionsPersonal` | 所得控除額（概算：基礎控除・配偶者控除・社保控除等の合計） |
| `directorSalaryAnnual` | 役員報酬（年額） |
| `corpMaintenanceCost` | 法人化による追加コスト（年額） |
| `age` | 年齢 |
| `isTaxableBusiness` | 個人事業税の対象業種フラグ |
| `isRentingHome` | 代表が賃貸住宅に住んでいるかフラグ |
| `monthlyRent` | 月額家賃（役員社宅制度用） |

## 2. 個人事業の計算

### 2-1. 事業所得

```
売上 - 経費 = 青色控除前の事業所得
青色控除 = min(青色申告特別控除額, max(青色控除前の事業所得, 0))
事業所得 = max(青色控除前の事業所得 - 青色控除, 0)
```

### 2-2. 個人事業税

対象業種の場合のみ計算（非対象なら 0円）。

```
課税標準 = max(事業所得 - 2,900,000, 0)
個人事業税 = floor(課税標準 × 5%)
```

### 2-3. 所得税

#### 課税所得

```
課税所得 = max(事業所得 - 所得控除額, 0)
```

#### 所得税の速算表

| 課税所得 | 税率 | 控除額 |
|---|---|---|
| 〜195万円 | 5% | 0円 |
| 〜330万円 | 10% | 97,500円 |
| 〜695万円 | 20% | 427,500円 |
| 〜900万円 | 23% | 636,000円 |
| 〜1,800万円 | 33% | 1,536,000円 |
| 〜4,000万円 | 40% | 2,796,000円 |
| 4,000万円超 | 45% | 4,796,000円 |

#### 計算式

```
所得税（基本） = floor(課税所得 × 税率 - 控除額)
復興特別所得税 = floor(所得税（基本） × 2.1%)
所得税 合計   = 所得税（基本） + 復興特別所得税
```

### 2-4. 住民税

```
所得割 = floor(課税所得 × 10%)
均等割 = 5,000円
住民税 合計 = 所得割 + 均等割
```

### 2-5. 国保・国民年金

手動入力がある場合はその値を使用。自動計算の場合：

```
国民年金 = 17,510円 × 12ヶ月 = 210,120円
国保（概算） = floor(max(事業所得 - 430,000, 0) × 10%)
国保上限 = 1,060,000円
国保・国民年金 合計 = 国民年金 + min(国保概算, 国保上限)
```

### 2-6. 個人の手取り

```
手取り = 事業所得 - 所得税合計 - 住民税合計 - 個人事業税 - 国保・国民年金合計
```

## 3. 法人化後の計算（個人側）

### 3-1. 給与所得控除

役員報酬を給与所得に変換する。

| 役員報酬（年額） | 控除額 |
|---|---|
| 〜162.5万円 | 55万円（固定） |
| 〜180万円 | 収入 × 40% - 10万円 |
| 〜360万円 | 収入 × 30% + 8万円 |
| 〜660万円 | 収入 × 20% + 44万円 |
| 〜850万円 | 収入 × 10% + 110万円 |
| 850万円超 | 195万円（上限） |

```
給与所得 = max(役員報酬 - 給与所得控除, 0)
```

### 3-2. 所得税（法人化後・個人）

```
課税所得 = max(給与所得 - 所得控除額, 0)
所得税（基本） = floor(課税所得 × 税率 - 控除額)   ※速算表は個人事業と同じ
復興特別所得税 = floor(所得税（基本） × 2.1%)
所得税 合計   = 所得税（基本） + 復興特別所得税
```

### 3-3. 住民税（法人化後・個人）

```
所得割 = floor(課税所得 × 10%)
均等割 = 5,000円
住民税 合計 = 所得割 + 均等割
```

### 3-4. 社会保険料

手動入力がある場合はその値を使用。自動計算の場合（協会けんぽ・東京都 概算）：

```
月額報酬 = floor(役員報酬 ÷ 12)
```

#### 健康保険

```
健保対象月額 = min(月額報酬, 1,390,000)
健保料率 = 9.98%
  ※40歳〜64歳の場合：+1.60%（介護保険料）= 11.58%
健康保険（年額） = floor(健保対象月額 × 健保料率) × 12
```

#### 厚生年金

```
厚年対象月額 = min(月額報酬, 650,000)
厚生年金（年額） = floor(厚年対象月額 × 18.30%) × 12
```

#### 合計・折半

```
社保合計 = 健康保険（年額） + 厚生年金（年額）
本人負担 = floor(社保合計 ÷ 2)
会社負担 = 社保合計 - 本人負担
```

### 3-5. 役員社宅制度

代表が賃貸住宅に住んでいる場合、法人化後に役員社宅制度を活用できる。

```
年間家賃 = 月額家賃 × 12
法人負担分（損金） = floor(年間家賃 × 50%)
個人の実質手取り増 = 法人負担分（給与課税なし）
```

- 法人が家賃の50%を負担し、損金算入できる
- 社長の自己負担は家賃の約50%
- 法人負担分は社長への給与課税の対象にならない
- 個人事業のままの場合、家賃は全額プライベート支出（必要経費として処理済み）

### 3-6. 法人化後の個人手取り

```
個人の手取り = 役員報酬 - 所得税合計 - 住民税合計 - 社保（本人負担） + 役員社宅による実質手取り増
```

## 4. 法人化後の計算（会社側）

### 4-1. 税引前利益

```
税引前利益 = (売上 - 経費) - 役員報酬 - 法人化追加コスト - 社保（会社負担） - 役員社宅（法人負担分）
```

> ※ 青色申告特別控除は個人事業のみの制度のため、法人側では適用されません。

### 4-2. 法人税（国税）

中小法人（資本金1億円以下）の税率を適用。

```
課税所得 = max(税引前利益, 0)
```

| 課税所得 | 税率 |
|---|---|
| 800万円以下の部分 | 15% |
| 800万円超の部分 | 23.2% |

```
法人税 = floor(800万円以下部分 × 15% + 800万円超部分 × 23.2%)
```

### 4-3. 地方法人税（国税）

法人税額をベースに計算。

```
地方法人税 = floor(法人税 × 10.4%)
```

### 4-4. 法人住民税（地方税）

#### 法人税割

```
法人住民税（法人税割） = floor(法人税 × 7.0%)
```

#### 均等割

資本金1,000万円以下・従業員50人以下の最低ランク。

```
法人住民税（均等割） = 70,000円（固定）
```

> ※ 赤字でも均等割は発生します。

```
法人住民税 合計 = 法人税割 + 均等割
```

### 4-5. 法人事業税（地方税）

所得割の超過累進税率。

| 課税所得 | 税率 |
|---|---|
| 〜400万円 | 3.5% |
| 〜800万円 | 5.3% |
| 800万円超 | 7.0% |

```
法人事業税 = floor(各区分の所得 × 各税率 の合計)
```

### 4-6. 特別法人事業税（地方税）

法人事業税額をベースに計算。

```
特別法人事業税 = floor(法人事業税 × 37%)
```

### 4-7. 法人税等 合計

```
法人税等 合計 = 法人税 + 地方法人税 + 法人住民税合計 + 法人事業税 + 特別法人事業税
```

### 4-8. 税引後利益（内部留保）

```
税引後利益 = 税引前利益 - 法人税等合計
```

---

## 5. 比較・判定

### 5-1. 合算キャッシュ

```
合算キャッシュ = 法人化後の個人手取り + 税引後利益
```

### 5-2. 差額

```
個人手取りベースの差額 = 法人化後の個人手取り - 個人事業の手取り
合算キャッシュベースの差額 = 合算キャッシュ - 個人事業の手取り
```

### 5-3. 判定ロジック

合算キャッシュベースの差額で3段階に判定。

| 条件 | 判定 |
|---|---|
| 差額 ≧ +30万円/年 | **法人化が有利な可能性が高い** |
| -30万円 < 差額 < +30万円 | **条件次第（ボーダーライン）** |
| 差額 ≦ -30万円/年 | **まだ法人化は早い可能性** |

### 5-4. 判定理由の自動生成

以下の差分要因を絶対値でソートし、上位3つを表示。

| 要因 | 計算式 |
|---|---|
| 所得税の差 | 個人の所得税 - 法人化後の所得税 |
| 住民税の差 | 個人の住民税 - 法人化後の住民税 |
| 社保の差 | 国保年金 -（社保本人負担 + 社保会社負担） |
| 法人税等の負担 | -法人税等合計 |
| 個人事業税の削減 | +個人事業税（法人化でなくなる） |
| 青色控除の消失 | -青色申告特別控除額（法人化で使えなくなる） |
| 法人化追加コスト | -追加コスト |
| 役員社宅制度の活用 | +家賃の50%（法人経費化＋個人手取り増） |
| 内部留保 | +税引後利益 |

## 注意事項

- 本ツールは **概算シミュレーション** です。実際の法人化判断には個別の詳細試算が必要です。
- 税率は **2026年度の標準税率（概算）** を使用しています。自治体や業種により異なる場合があります。
- 社会保険料は **協会けんぽ（東京都）の概算料率** で計算しています。
- **消費税の影響** は本シミュレーションに含まれていません。
- 法人住民税の均等割は **資本金1,000万円以下・従業員50人以下** の最低ランクで計算しています。
